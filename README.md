What does it do :question:
===========================
BFKit is a collection of useful classes to **develop Apps faster**.

For example you can use every iOS font with just an enum!

It also adds some useful method to some classes like:

###BFKit:
- [BFApp
- BFButton
- BFLog
- BFPassword
- BFSystemSound
- BFTouchID

###Foundation:
- Array
- Dictionary
- NSArray
- NSDate
- NSDictionary
- NSFileManager
- NSMutableArray
- NSMutableDictionary
- NSNumber
- NSObject
- NSProcessInfo
- NSString
- NSThread
- String
